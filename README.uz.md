## Installation

You can install the package via composer:

```bash
composer require goodoneuz/pay-uz
```

## Usage
------
- Request handle: `PayUz::driver('payme')->redirect($model, $amount, $currency)`
- Redirect user:  `PayUz::driver('payme')->handle()`

------
